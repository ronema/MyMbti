# MyMbti - 性格测试小程序

## 项目介绍
MyMbti 是一款基于 MBTI 理论的性格测试小程序，帮助用户了解自己的性格类型和特点。

## 功能特点
- 16种MBTI性格类型测试
- 精准的性格分析
- 现代化的用户界面
- 个性化的结果展示

## 技术栈
- 微信小程序
- JavaScript
- WXML/WXSS

## 版本
当前版本：1.3.0

## 使用说明
1. 打开微信小程序
2. 进入"MBTI测试"
3. 回答93个问题
4. 查看你的性格类型
